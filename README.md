# MyPeople Hubot Adapter

## Usage

ë

```shell
export HUBOT_MYPEOPLE_KEY='API KEY'
vi package.json
...
  "dependencies": {
    "hubot":         ">= 2.5.1",
    "hubot-scripts": ">= 2.4.2",
    "hubot-mypeople": "0.0.2",
    "express": ">=3.0.0"
  },
...

bin/hubot -a mypeople
```

[MyPeople API](https://github.com/dgkim84/node-mypeople)
